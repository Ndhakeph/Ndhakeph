## 🚀 Featured Projects

### 🛡️ [AI Safety Testing Harness](https://github.com/tacitusblindsbig/ai-safety-harness)
Production-ready red-team testing platform for evaluating LLM safety mechanisms
- **30+ adversarial prompts** across 5 attack categories (jailbreaks, injection, harmful content, etc.)
- **Multi-layer guardrail detection** with automated safety scoring
- **Tech:** Python, FastAPI, PostgreSQL, Next.js, TypeScript
- **Impact:** 95% detection accuracy on jailbreak attempts

### 📊 [AI Evaluation Platform](https://github.com/tacitusblindsbig/ai-eval-platform)
Automated quality assessment for AI-generated content using LLM-as-Judge
- **3-dimensional rubric** (Accuracy, Clarity, Completeness) with 0-10 scoring
- **Batch processing** via CSV upload for scalable evaluation
- **Tech:** Next.js, TypeScript, Supabase, Google Gemini API
- **Quality:** 0.87 correlation with human ratings

### 🤖 [AI Content Pipeline](https://github.com/tacitusblindsbig/ai-content-pipeline)
Multi-agent system for automated blog post generation with fact-checking
- **4 specialized agents** (Researcher→Writer→Fact-Checker→Polisher)
- **Automated revision loops** with retry logic for quality assurance
- **Tech:** Next.js, TypeScript, Gemini API, Tavily Search
- **Performance:** 96% factual accuracy, $0.004 per article

### 🎯 [RAG Knowledge Assistant](https://github.com/tacitusblindsbig/ai-rag-knowledge-assistant)
Fully local RAG system with 100% offline inference using Ollama
- **Zero API dependencies** - Complete privacy with local LLMs
- **Production-grade** - HNSW vector search, streaming responses
- **Tech:** Next.js, LangChain.js, Ollama (Gemma3), Supabase pgvector
- **Speed:** Sub-3s query times with semantic search
